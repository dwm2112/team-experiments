# Channel Plan: Atlas Sovereign Agentic Kickoff

**Experiment:** `experiments/002-marketing_team_workflow/`  
**Role:** growth-marketer  
**Pipeline:** marketing-client · stage: campaign_design  
**Board status:** Direction approved only (SMB-primary · trustworthy digital coworker · LinkedIn + founder-led + waitlist). **Not** locked for outbound, media spend, package pricing, or numeric funnel commits until client confirmation package clears + human nurture approve.  
**North Star:** Qualified waitlist signup → books discovery (or accepts calendar invite) within **≤14 days**.  
**Execution owner (post-gates):** Founder (or designated closer) runs LinkedIn + discovery; client hosts waitlist page from package specs.

---

## Channel mix & rationale

| Priority | Channel / motion | Role in funnel | Why this bet |
|----------|------------------|----------------|--------------|
| **P0** | Founder-led LinkedIn (organic) | Awareness → Acquisition | Near-zero media means demand is founder-authority + cadence. ICP (SMB ops/owner/COO) lives here; claim-safe coworker stories travel better as founder POV than brand ads. |
| **P0** | Founding-partner waitlist (client-hosted page) | Acquisition → Activation | Single conversion surface; ICP-qualify on form; UTM’d from every LinkedIn CTA. Design package specs page; **client ships/hosts**. |
| **P0** | 3-touch nurture (email **or** LinkedIn DM — pick one primary) | Activation → Revenue | Converts waitlist → discovery under North Star; qualification gate before booking burns scarce calendar. |
| **P1** | Discovery calls (founder calendar) | Revenue (to SQL) | Bottleneck is show-rate + capacity, not impressions. Kickoff metric stops at SQL / scoped-pilot handoff. |
| **P2 (optional)** | LinkedIn boost **ASSUMPTION ≤$500** | Acquisition assist | Only after 1–2 organic posts prove ICP engagement; never always-on Meta/Google. |
| **P2 (supporting)** | Minimal content spine (1 cornerstone page + 3–4 pillar posts) | Share permanence / SEO assist | Board change: thin LinkedIn burst without a spine dies after week 6. Spec + copy in package; client may publish on site/blog. |
| **Defer** | Warm intros / waitlist reshare asks | Referral | Ask on every discovery; do not build a referral program in kickoff. |
| **Out** | Multi-division brand (Robotics/Intelligence), spray paid, multi-ICP creative | — | Dilutes thin signal; board locked Agentic AI Division + SMB-primary. |

**Working hypothesis (ASSUMPTION):** A claim-safe founder cadence of **3–5 LinkedIn posts/week + daily reply window** can generate enough ICP-fit traffic for a qualified waitlist→discovery loop without paid, *if* waitlist + booking instrument + ≤48h SLA are live.

**Soft ramp (board change):** Treat weeks 1–3 as demand/instrument proof; press North Star hard in weeks 4–6 only if cadence + qualify definition + calendar hold.

## Experiments / tests (max 3, cheap)

Run only after gates clear. Cap total incremental cash at media ASSUMPTION band.

### Test 1 — CTA placement on founder posts (free)

- **Hypothesis:** Link-in-comments + plain-language coworker CTA beats “DM me” for waitlist volume without looking spammy.
- **Variants:** (A) link in first comment within 5 min, (B) “comment WAITLIST / I’ll send link” (use only if spam filters or policy prefer).
- **Metric:** Waitlist submits with `utm_content` per variant; % qualified.
- **Duration:** 2 weeks / ≥6 posts.
- **Kill:** If B creates DM backlog > founder can clear in ≤48h → stick to A.

### Test 2 — Qualify-form friction (free)

- **Hypothesis:** One intent question (“I need a coworker for a scoped workflow” vs “I want AI to replace headcount”) raises qualified→discovery without crushing signup volume >40%.
- **Variants:** Short form (role + size + pain) vs short + intent filter.
- **Metric:** Signup volume, qualified %, discovery book rate ≤14d.
- **Duration:** Week 2–3 (switch once; don’t thrash weekly).
- **Kill:** If volume drops >40% and absolute qualified discoveries fall → revert friction, tighten nurture instead.

### Test 3 — Single LinkedIn boost on winner post (≤$250 first shot, from $0–500 ASSUMPTION)

- **Hypothesis:** Boosting the best organic ICP-engagement post adds qualified waitlist cheaper than boosting a cold creative.
- **Setup:** Organic run ≥5 days; boost only if comments/saves from Owner/COO/Ops titles; ICP title targeting; 48–72h cap.
- **Metric:** Incremental qualified waitlist; $/qualified signup; discovery book rate of boosted cohort.
- **Kill:** If >50% boosted signups fail qualify gate or $ / qualified discovery > organic opportunity cost → stop paid; do not spend remainder on second boost.

**Do not run in kickoff:** multi-geo ad sets, lead-gen forms that bypass waitlist qualify fields, Meta/Google always-on, A/B on autonomy-forward claims.

## Kill / pivot triggers

| Trigger | Action |
|---------|--------|
| Client confirmation package incomplete (budget/dates/claim reviewer/cadence/waitlist host/calendar/SOR) | **No outbound, no pricing lock, no boosts** |
| Client refuses claim lexicon / insists “autonomous digital employees” in market copy | Pause public creative; reopen board / re-scope |
| Founder cadence miss: <3 posts/week **or** no reply block for **2 consecutive weeks** | Pause waitlist push; fix operating rhythm before volume goals |
| Two consecutive cadence-OK weeks with weak **qualified** waitlist despite non-trivial impressions | Reject/pause organic-primary; options: warm pilot path, raise media band, or narrow offer |
| Qualified → discovery **<25%** by end of Week 3 (instruments live) | Fix nurture/SLA/booking before more TOFU; revoke treating funnel bands as working targets |
| Discovery capacity **<2 slots/week** or no closer | Drop numeric funnel bands; sales-led only / delay launch |
| Dual measurement SORs or undefined MQL | Freeze reporting claims; pick one SOR before any “success” narrative |
| Boost cohort >50% disqualified or toxic CPL | Kill paid remainder; return to organic |
| Multi-division or replace-your-team messaging leaks into live posts | Immediate content freeze + claim review |
| Same blocker twice (e.g. calendar still undefined after two asks) | Escalate to human; do not re-loop stage |
